# Kids Letter Quiz

A lightweight alphabet quiz for practicing letters, words, and sounds.

Live site: https://tw0316.github.io/Kids-letter-quiz/

## Features

- Big letter + emoji cards for young kids
- Correct / incorrect scoring
- Progress through the alphabet
- Tap the card or press **Say It** to hear the letter phrase
- Optional auto-speak
- Optional lowercase mode
- Optional random order
- Missed-letter retry and missed-only practice
- Reset button
- Mobile-safe layout for phones and tablets
- Installable/offline-friendly GitHub Pages app via web manifest + service worker

## Keyboard shortcuts

- **Right arrow:** correct
- **Left arrow:** incorrect
- **Space / Up arrow:** say it
- **R:** reset
